# ATS Recruit

Sistema web de reclutamiento (ATS) desarrollado en HTML, CSS, JavaScript y Firebase.

## 🔧 Tecnologías
- HTML5
- CSS3
- JavaScript (Vanilla)
- Firebase (Auth + Realtime DB)

## 📁 Estructura
- /candidatos → perfil y CV
- /recruit → panel reclutador
- /js → lógica del sistema
- /css → estilos

## 🚀 Funcionalidades
- Registro e inicio de sesión
- Perfil de candidato
- Generación de CV dinámico
- Gestión de vacantes
- Panel de reclutador

## 📌 Pendientes
- Multiempresa
- Roles avanzados
- Dashboard con métricas
